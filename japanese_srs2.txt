**백엔드 `/srs/available` 엔드포인트**가 언어별 분리 시스템 구현으로 **새로운 응답 구조**로 변경됨:

#### 기존 구조 (구 버전)
```javascript

  data: [card1, card2, card3]  // 배열 형태
}
```

#### 새로운 구조 (신 버전)
```javascript
{

  data: {
    japanese: [card1],         // 일본어 카드들
    english: [card2, card3],   // 영어 카드들
    total: 3,                  // 전체 카드 수
    hasMultipleLanguages: true // 혼합 언어 여부
  }
}
```

### 3. 코드 동기화 문제
